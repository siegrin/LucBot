## How To Run

<details>
  <summary>TERMUX/UBUNTU/SSH USER</summary>
	
  ```bash
apt update && apt upgrade
apt install git -y
apt install nodejs -y
apt install ffmpeg -y
git clone https://github.com/siegrin/LucBot
cd LucBot
bash install.sh
pkg install yarn
yarn
```
</details>

  <summary>WINDOWS/RDP</summary>
